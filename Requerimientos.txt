PANTALLA RUTINA para crear, editar o eliminar RUTINA que contenga los campos
 - Año.- se debe seleccionar un año
 - Mes.- se debe seleccionar el mes
 - Dia.- se debe seleccionar el dia
 - Hora Inicio.- se debe seleccionar hora de inicio
 - Hora fin.- se debe seleccionar hora de fin 
 - Rutina.- campo para ingresar la rutina de la clases
 - Imagen.- imagne de la rutina 
 - Entrenador.- se debe guardar automaticamente el nombre del entrenador que esta creando la Rutina, este campo no es editable
 - Estado .- puede ser en espera, rechazada, aprobada; este campo no es editable 
 - Se debe visualizar en una lista todas las RUTINA y debe tener una colimna de acciones para editar o eliminar
	debe existir un boton para crear una nueva.

REGLAS DE NEGOCIO
PANTALLA CLIENTE 
- Cuando pasa la fecha de fin de menbresia automaticamente el cliente se debe desactivar.
LOGIN 
- Solo debe permitir ingresar a SUPER ADMINISTRADOR que se encuentren activos
- Solo debe permitir ingresar a ADMINISTRADOR que se encuentren activos
- Solo debe permitir ingresar a EDECAN  que se encuentren activos
- Solo debe permitir ingresar a ENTRENADOR  que se encuentren activos
- Solo debe permitir ingresar a CLIENTES  que se encuentren activos
- El usuario debe ser la cedula, u el campo solo debe permitir el ingreso de numeros.
- La contraseña debe ser alfanumerica, una letra mayuscula, letra minuscula, numero.
- El numero de intentos de ingreso debe ser hasta tres si sobrepasa se desactiva la cuenta 
PANTALLA GIMNASIO
- Si un gimnasio esta desactivado todos los ADMINISTRADOR,EDECAN, ENTRENADOR se deben desactivar automaticamente
PANTALLA RUTINA
- Un entrenados no puede crear dos rutinas en horarios que interfieran, si hace esto debe mostrar el mensaje no se puede crear rutinas en el mismo horario
- Rutina debe ser aprobada por el ADMINISTRADOR O EDECAN
- Una rutina aprobada no puede ser eliminada o desactibada
PANTALLA USUARIO 
- Solo el administrador general puede seleccionar los gimnacion
- Si el usuario que esta usando el sistema pertenese a un gimacion el campo 
	Gimnacio de seleccionarse automaticamente con el mismo gimnasio del usuario que esta utilizando y no se puede modificar
PANTALLA CLIENTE
-Si el usuario que esta usando el sistema pertenese a un gimacion el campo 
	Gimnacio de seleccionarse automaticamente con el mismo gimnasio del usuario que esta utilizando y no se puede modificar
PANTALLA REGISTRO RUTINA
- El ADMINISTRADOR puede ver la PANTALLA REGISTRO RUTINA pero no se puede registrar
- El EDECAN puede ver la PANTALLA REGISTRO RUTINA pero no se puede registrar
- Solo se muestran rutinas aprobadas
